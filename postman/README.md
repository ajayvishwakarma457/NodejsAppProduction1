# Postman

Import `NodejsAppProduction1.postman_collection.json` into Postman.

Default collection variable:

- `baseUrl=http://localhost:4000`

Available requests:

- `GET /health`
- `POST /api/v1/auth/login`
- `GET /api/v1/users`
- `GET /api/v1/teams`
- `GET /api/v1/projects`
- `GET /api/v1/tasks`
- `GET /api/v1/comments`
- `GET /api/v1/notifications`
