# AI Context Folder

Use this folder to store project context that AI tools can read before making changes.

Recommended workflow:

1. Put stable project information in `project-overview.md` and `architecture.md`.
2. Put data models and relationships in `models.md`.
3. Put changing information in `current-status.md` and `tasks.md`.
4. Record important decisions in `decisions.md`.
5. Keep entries short, factual, and updated.
