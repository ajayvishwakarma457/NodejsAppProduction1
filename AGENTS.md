# AI Working Guide

This repository keeps AI-readable project context in `/ai-context`.

Read these files first:

1. `/ai-context/project-overview.md`
2. `/ai-context/current-status.md`
3. `/ai-context/architecture.md`
4. `/ai-context/models.md`
5. `/ai-context/decisions.md`
6. `/ai-context/tasks.md`

Rules for AI tools:

- Keep changes small and explicit.
- Update context files when major decisions or architecture changes happen.
- Prefer adding facts from the real codebase instead of assumptions.
- Record open questions in `/ai-context/tasks.md`.
